# comfysd
这是在模仿sdwebui做出来的，用来连接本地或远程comfyui（本人用的autodl，实测可行），向它发送存在同目录下workflows内的工作流api文件，以达到生图效果的网站。
用于本地的将在此工程完成后单独适配并单独发布。

只推荐有编程基础和web基础的人使用。或者至少会问ai。。。

一键启动服务可以选择”startWithAutodl.bat“，启动服务时顺带启动云端实例，本地也可以，但是要修改代码。
直接启动服务不启动autodl（或本地comfyui）选择”start.bat“即可。
